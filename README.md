# 🤖 Jarvis AI Assistant

## 📌 Description

Jarvis AI is a simple virtual assistant built to perform basic computer tasks using voice or text commands. It helps users interact with their system more easily by automating actions like opening applications, searching the web, and handling simple operations.

The main focus of this project is usability and clarity. The assistant listens to user input, processes it, and responds with the appropriate action. It is designed in a way that even beginners can understand and modify the code without difficulty.

This project demonstrates the basic concept of automation and how a command-based assistant can improve productivity. It also serves as a learning example for those interested in building similar AI-based tools.

## 🚀 Features

* Voice and text command support
* Opens applications and websites
* Simple and user-friendly

## 🛠️ Technologies Used

* Python
* Speech Recognition
* Text-to-Speech (TTS)
